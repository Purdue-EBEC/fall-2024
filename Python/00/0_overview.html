0. Welcome to Class#

Unit 00 of our course focuses on getting your system ready for the rest of the semester. Your main objectives for this unit are:

  1. Access course resources:

    • Brightspace: course content

    • Campuswire: discussion and Q&A forum

    • Gradescope: exercise submission and grading

    • Textbook: Python Crash Course 3rd edition

    • Zoom: team meetings and office hours

  2. Install necessary software: (see the Software Installation Steps for your system).

    • Python 3.12: interprets and runs your code

    • Visual Studio Code: tool for writing code

  3. Complete assignments:
